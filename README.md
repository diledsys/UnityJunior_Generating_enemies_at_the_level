# Unity Project: Basic Movement

## Описание
Учебный проект на Unity.
На уровне присутствует несколько точек спавна (создания врагов).
Каждые две секунды в случайной точке спавна должен появляться новый враг. После этого он должен начать двигаться по прямой в том направлении, которое ему укажет спавнер.

## Используемые технологии
- Unity 6.3
- C#

## Структура
- Assets/_Project/Scripts — логика управления
- Assets/_Project/Scenes — основная сцена
- Assets/_Project/Prefab

## SpawnPoint
- Назначение: точка появления врагов + источник направления движения
- Отвечает за:
- позицию спавна (Position)
- направление движения (Direction как Vector3)
- визуальную отладку направления (Gizmos)
- Не делает:
- не спавнит врагов
- не двигает врагов
- не знает про таймеры и пул

## EnemySpawner
- Назначение: управление процессом спавна
- Отвечает за:
- периодичность спавна (корутина + интервал)
- выбор случайного SpawnPoint
- запрос врага из пула
- передачу позиции и направления врагу
- Не делает:
- не создаёт объекты напрямую (Instantiate)
- не управляет движением врага
- не уничтожает врагов

## EnemyPool
- Назначение: повторное использование врагов (memory-friendly)
- Отвечает за:
- хранение неактивных врагов
- выдачу активного врага
- возврат врага обратно в пул
- Не делает:
- не знает, зачем враг появился
- не знает, куда он идёт
- не считает время жизни

## EnemyMover
- азначение: движение и ориентация врага
- твечает за:
- вижение по заданному направлению
- оворот лицом к направлению
- инхронизацию скорости движения с анимацией
- становку параметра Speed в Animator
- е делает:
- е спавнит себя
- е уничтожает себя
- е знает про пул

## ReturnToPoolAfterTime
- Назначение: ограничение жизни врага
- Отвечает за:
- отсчёт времени жизни
- возврат врага в пул по таймеру
- Не делает:
- не создаёт врага
- не управляет его движением
- не решает, когда он появился

## Как запустить
1. Открыть проект в Unity Hub
2. Загрузить сцену Main
3. Нажать Play
