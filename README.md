# Unity Project: Basic Movement

## Описание
Учебный проект на Unity.
Реализовано управление объектом, движение и базовая логика.

## Используемые технологии
- Unity 2022.6
- C#

## Структура
- Assets/Scripts — логика управления
- Assets/Scenes — основная сцена

## Как запустить
1. Открыть проект в Unity Hub
2. Загрузить сцену Main
3. Нажать Play
